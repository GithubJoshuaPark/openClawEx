Intro

1. OpenClaw란?

AI 에이전트를 운영할 수 있는 최적의 플랫폼입니다. Claude, OpenAI 등 최신 모델과 연동하여 자동화 작업을 수행합니다.

  • ✨ AI 자동 메시지 및 작업 실행
  • 💬 WhatsApp, Telegram, Discord 연동
  • 🔍 웹 검색 및 실시간 정보 처리
Step 01

2. Node.js 버전 확인

OpenClaw는 Node.js v22.14.0 이상이 필요합니다.

# nvm으로 v24.7.0 권장 버전 활성화
nvm use 24.7.0
nvm alias default 24.7.0
Step 02

3. OpenClaw 설치

간단한 쉘 명령어로 패키지를 설치하고 초기화할 수 있습니다.

# 설치 스크립트
curl -fsSL https://openclaw.ai/install.sh | bash

# 확인 및 설치 초기화
openclaw --version
openclaw setup

Troubleshooting

10. 트러블슈팅 가이드

🔴 Node.js 버전 오류

시스템 심볼릭 링크를 nvm 경로로 수정하여 해결합니다.

sudo ln -sf ~/.nvm/versions/node/v24.7.0/bin/node /usr/local/bin/node

🔴 WebSocket 4014 에러

Discord Intents 권한 미설정 시 발생합니다. Developer Portal에서 3가지 옵션을 활성화하세요.

Daily Routine

14. 맥북 재부팅 후 빠른 시작

매번 입력하는 것이 번거롭다면 ~/.zshrc에 별칭(Alias)을 등록하세요.

# ~/.zshrc Alias 추천
alias claw='source ~/.nvm/nvm.sh && nvm use 24.7.0 && openclaw gateway start'
alias clawchat='source ~/.nvm/nvm.sh && nvm use 24.7.0 && openclaw'
