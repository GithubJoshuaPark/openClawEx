Introduction

OpenClaw란?

AI 에이전트를 운영·관리할 수 있는 오픈소스 플랫폼

💬

Channel

Discord Bot 연동

Discord Developer Portal에서 봇을 만들고 OpenClaw에 연결

  1. Discord Developer Portal 접속
  2. "New Application" → 이름 입력 (예: OpenClaw Bot)
  3. 좌측 "Bot" → "Reset Token" → 토큰 복사
🔐 토큰은 한 번만 표시됩니다! 안전하게 보관하세요.

Bot 페이지 하단 "Privileged Gateway Intents"에서 3가지 모두 활성화:

  • ✅ PRESENCE INTENT
  • ✅ SERVER MEMBERS INTENT
  • ✅ MESSAGE CONTENT INTENT
❌ 미설정 시 에러 4014 발생 — 봇이 메시지를 읽지 못합니다.

채널 ID 복사: Discord 앱 → 설정 → 고급 → 개발자 모드 ON → 채널 우클릭 → 채널 ID 복사

서버 초대: OAuth2 → URL Generator → scope: bot 선택 → 생성된 URL로 초대

Execution

Gateway vs Terminal 모드

두 가지 실행 모드를 상황에 맞게 사용하세요

🌐
Gateway 모드
백그라운드
openclaw gateway start
  • 터미널 닫아도 계속 동작
  • Discord에서 @Bot 멘션으로 대화
  • 모바일 푸시 알림 수신
  • 봇이 먼저 연락 가능
  • 대시보드: localhost:18789
⌨️
Terminal 모드
포그라운드
openclaw
  • 터미널 닫으면 세션 종료
  • Discord 없이 바로 대화
  • 파일 시스템 직접 접근
  • 코드 수정 / git 작업 즉시
  • 1회성 질문에 적합

Troubleshooting

트러블슈팅

실전에서 겪은 에러와 해결법

원인: /usr/local/bin/node가 옛 버전을 가리킴

nvm use 24.7.0 # 또는 영구 수정: sudo ln -sf ~/.nvm/versions/node/v24.7.0/bin/node /usr/local/bin/node

원인: Discord Developer Portal에서 Privileged Gateway Intents 미활성화

해결: Bot → Privileged Gateway Intents → 3가지 모두 ✅ → Save Changes → openclaw gateway restart

원인: openclaw.json에 지원하지 않는 키 추가됨

node -e " const fs = require('fs'); const p = process.env.HOME+'/.openclaw/openclaw.json'; const c = JSON.parse(fs.readFileSync(p)); delete c.channels.discord.defaultChannel; fs.writeFileSync(p, JSON.stringify(c,null,2)); " openclaw gateway restart

원인: LaunchAgent가 nvm 경로를 직접 참조, 업그레이드 시 깨질 수 있음

openclaw doctor --repair
nvm 의존 없이 안정적인 경로로 LaunchAgent를 재설정합니다.

원인: gemini-1.5-flash 등 구 버전 모델 Deprecation

# openclaw.json에서 모델 업데이트 { "primaryModel": "google/gemini-2.5-flash", "activeModel": "google/gemini-2.5-flash" }
Quick Start

맥북 재부팅 후 빠른 시작

매번 입력할 필요 없이 Alias로 한 번에!

⌨️ 수동 시작
# 1. nvm 활성화 source ~/.nvm/nvm.sh # 2. Node.js 전환 nvm use 24.7.0 # 3. Gateway 시작 openclaw gateway start
💡 Pro Tip: Alias 설정 추천
# ~/.zshrc 에 추가 alias claw='source ~/.nvm/nvm.sh && nvm use 24.7.0 && openclaw gateway start' alias clawchat='source ~/.nvm/nvm.sh && nvm use 24.7.0 && openclaw'
단축어 한 번으로 환경 설정과 실행을 동시에 처리합니다.
